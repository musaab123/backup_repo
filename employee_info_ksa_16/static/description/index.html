Overview

This module added some advanced features and extra fields on Employee master. Such as dependents informations, education informations, joining date, national/iqama id with passport and expiry date for each document as they can upload an image attachements.

This module also sends expiry notification for passport and national/iqama id to the employees.

Features

  • Insert National/Iqama ID with expiry date & upload a copy as attachments.
  • Insert dependence details with important informations as (Name, National/Iqama ID, Relation... etc).
  • Insert National Address Details with uploading verification certificate.
  • Add another important information like passport, education and additional detils.
  • Send auto reminder for National/Iqama and Passport before expiry date.
